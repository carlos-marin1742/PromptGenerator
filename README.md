# PromptGen

Local AI prompt generator — runs as a Node.js server, opens in your browser.

## Setup

```bash
npm install
npm start
```

Opens automatically at http://localhost:3737

## Every time after that

```bash
npm start
```

## Store location

Settings and aliases are saved to:
```
C:\Users\<YourName>\.promptgen-store.json
```
